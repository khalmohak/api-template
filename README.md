This is a Node Js boiler plate that I commonly use for my projects.
Babel is setup here, so you can use ES6 without any worries. 
'db' folder is for SQL files
You can easily set up your Postgres database connection in the database.js file.
app.js contains all the routes.



To start the Project:
1. Set-up the connection string of Postgres in database.js file
2. Run yarn install to install the Packages associated with this project.
3. Run the project using yarn start and check the routes.
